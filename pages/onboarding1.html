Introduction

This is a user test. I am collecting user data to prove a hypothesis in my dissertation. The test is aiming to collect data on user visual attention in a digital interface. There are three tasks, each slightly different on three different pages.

Each page represents a real website just like any other website you may visit to pay a bill or check your bank balance. Each task will ask you to perform a specific action on the webpage.

This test is not aiming to establish your ability, it is aiming to understand how users attention is directed using several techniques.

Each page will have an instruction on what action to carry out. The instruction will be at the top left of each page, once you have read through the instruction please carry out the task in your own time.

Whilst duration is not important we are looking at which element on the page you click first to complete the task, so please click the first button you see.

The test will only take a few minutes and will not track or save any personal information. Thank you so much for your time.
